Haste

Java server framework for multiplayer games

Overview

Haste is a multiplayer game server framework rather than a general purpose network library. It offers many features which a multiplayer game needs such as security, various QoS, and game-room.

Features

  • Various Quality of Service for the mulitplayer game.
  • Multiplexed stream transport over UDP without head of line blocking.
  • Data Encryption and Decryption with one-time secure key
  • Seemless handover from Wifi to Cellular network.
  • Message based concurrency similar to event based actors in Scala
  • Lock-free memory pool

Future features

  • C++ SDK
  • TCP protocol
